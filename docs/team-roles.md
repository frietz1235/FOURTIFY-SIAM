## Role Responsibilities Summary

### PM / Scrum Master
- Owns `backlog.md` and `sprint-*-plan.md`
- Facilitates daily standups
- Reports progress every Friday

### QA Lead
- Creates `test-cases.md` per sprint
- Logs bugs in `defect-log.md`
- Verifies fixes before closing issues

### DevOps Lead
- Ensures all code is committed with meaningful messages
- Sets up GitHub Actions (or Colab version control hooks)
- Manages environment variables for deployment

### Docs Lead
- Maintains `README.md`, `schedule.md`, `change-log.md`
- Ensures every PR updates relevant docs
- Prepares final submission documentation
